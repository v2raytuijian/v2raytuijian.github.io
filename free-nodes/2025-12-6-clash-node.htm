第一章 认识您的网络指挥官

1.1 Surfboard的技术基因

采用博通BCM4908四核处理器和512MB内存的硬件配置，使Surfboard系列在同时处理50+设备连接时仍能保持流畅。特别设计的六根高增益天线配合Beamforming波束成形技术，可将5GHz频段的覆盖范围提升40%，完美解决复式住宅的信号死角问题。

1.2 开箱清单核查指南

  • 主机本体（注意检查底部型号标签）
  • 12V/2A电源适配器（建议保留原装配件）
  • CAT6千兆网线（蓝色接口为WAN专用）
  • 多语言快速入门指南
  • 保修卡及服务二维码

专业提示：新型号SAX2-01已取消附赠网线，需提前准备超五类以上标准网线

第二章 物理连接的艺术

2.1 拓扑结构搭建

  1. 断电操作：先关闭光猫电源（防止IP冲突）
  2. WAN连接：使用附赠网线连接光猫LAN口与路由器蓝色WAN口
  3. 临时管理通道：用笔记本通过黄/灰色LAN口直连路由器（推荐初始配置阶段）
  4. 供电时序：先启动光猫，待所有指示灯稳定后再接通路由器电源

2.2 指示灯语言解读

| 指示灯颜色 | 状态模式 | 含义解析 |
|------------|----------|----------|
| 白色 | 常亮 | 系统正常启动 |
| 蓝色 | 闪烁 | 固件升级中 |
| 红色 | 呼吸灯 | WAN口未检测到信号 |
| 紫色 | 交替闪烁 | 恢复出厂设置中 |

3.1 管理界面登陆秘籍

3.2 双频段智能部署方案

2.4GHz频段配置：
- 信道宽度：20MHz（避免邻频干扰）
- 信道选择：使用WiFi Analyzer锁定1/6/11等非重叠信道
- 发射功率：调整为75%（平衡覆盖与辐射）

5GHz频段优化：
- 启用DFS信道（52-144频道）避开雷达干扰
- 开启802.11k/v/r协议实现无缝漫游
- 设置MU-MIMO设备优先策略

3.3 高级功能解锁

  1. QoS流量整形：为视频会议设备保留20%带宽
  2. 访客网络隔离：启用Client Isolation保护主网安全
  3. IPv6穿透设置：在"高级>网络设置"中启用DHCPv6

第四章 性能调优实验室

4.1 信号强度实测对比

使用Ekahau HeatMapper进行场地勘测：
- 最佳摆放高度：1.5-2米（避开金属柜体）
- 天线角度：45°倾斜创造立体覆盖
- 干扰源规避：距离微波炉至少3米

4.2 固件升级指南

  1. 官网下载页面验证型号（如SBG10对应文件SBG10-7.1.3G.img）
  2. 通过管理界面"维护>固件升级"上传文件
  3. 升级过程中保持不间断供电（建议连接UPS）

第五章 故障排除宝典

5.1 典型问题解决方案

情景1：5GHz信号消失
- 检查国家代码设置（需匹配所在地区）
- 关闭"智能连接"功能临时排查

情景2：NAT类型限制
- 启用UPnP并开放Xbox/PSN所需端口
- 设置DMZ主机前务必确认设备安全

5.2 工厂重置进阶技巧

长按WPS按钮10秒至所有LED同步闪烁，此时将保留最新固件仅清除配置数据，比传统复位孔方式更安全。

FAQ

VLESS + XTLS 模式在高并发下有什么优势？
XTLS 模式减少握手次数和加密冗余，提高 CPU 利用率和吞吐量。在高并发场景下能保持连接稳定，适合游戏、流媒体和频繁请求的网络环境，提升整体性能。
Surfboard 是否支持 Clash 配置？
Surfboard 是一款支持多种协议的客户端应用，它能够兼容 Clash 的配置文件格式，用户只需导入 YAML 文件即可。该应用以轻量、高速著称，常用于 iOS 和 macOS 系统。
Hiddify Next 用户流量配额如何设置？
管理员可为每个用户设置带宽和流量上限，超出限制可自动降速或阻止访问，保证服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性和使用效率。
Potatso Lite 在新版 iOS 上无法连接怎么办？
苹果系统更新后，Potatso Lite 可能需要重新授权 VPN 权限。请删除旧的 VPN 配置，重新安装并导入节点。若仍无法使用，可尝试更换网络模式或等待开发者更新兼容版本。
Potatso Lite 是否可导入节点二维码？
支持。用户可扫描节点二维码导入节点信息，也可手动输入节点参数。虽然不支持订阅自动更新，但对于轻量用户来说足够满足日常节点管理和流量分流需求。
V2Ray 如何通过策略组优化海外网站访问？
策略组可根据延迟和地理位置自动选择最佳节点访问海外网站。结合路由规则，可实现国内流量直连，国外流量通过代理，提高访问速度，降低延迟和连接不稳定风险。
Matsuri 支持订阅更新吗？
支持。Matsuri 内置订阅更新模块，可自动检测链接变化并定期刷新节点。用户还可手动点击更新按钮同步最新配置，保持网络连接的可用性与速度。
Mellow TUN 模式对游戏流量有何作用？
TUN 模式拦截系统 IP 层流量，实现全局代理。游戏流量通过低延迟节点传输，减少丢包和延迟，提高游戏体验的流畅性和网络稳定性。
Surfboard 支持哪些类型的节点导入？
Surfboard 支持 VMess、VLESS、Trojan、Shadowsocks 等节点类型。用户可通过 YAML 或 JSON 文件导入，配置策略组和规则，实现自动分流和节点管理。
PassWall2 如何定时自动更新节点？
可在计划任务中添加命令 `passwall update` 并设定时间间隔，例如每日凌晨自动更新。这样可确保节点信息实时同步，避免旧节点失效影响连接体验。
